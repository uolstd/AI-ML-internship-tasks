**Task 1: Exploring and Visualizing a Simple Dataset**
Objective:
 Learn how to load, inspect, and visualize a dataset to understand data trends and distributions. 
 **Dataset Used**
Iris dataset
Contains measurements of flowers:
Sepal length & width
Petal length & width
3 types of flowers (species)
**Key Results & Findings**
Petal measurements show clear differences between species
Sepal measurements are less distinct
Some outliers exist (especially in sepal width)
Data is mostly clean and well-structured
